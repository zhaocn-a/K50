# DynamicOasis-K50 与 K40 适配差异说明

## 概述

本文档详细说明 DynamicOasis 项目从红米K40适配到红米K50的具体变更内容。

---

## 硬件差异对比

| 参数 | 红米K40 | 红米K50 | 差异影响 |
|------|---------|---------|----------|
| 屏幕尺寸 | 6.67英寸 | 6.67英寸 | 无变化 |
| 屏幕类型 | AMOLED | 三星 AMOLED | 相同 |
| 分辨率 | 2400×1080 | **3200×1440 (2K)** | K50更清晰 |
| 挖孔位置 | **左侧挖孔** | **居中挖孔** | 关键差异 |
| 前置摄像头 | 2000万 | 2000万 IMX596 | 像素相同 |
| 电池容量 | 4520mAh | **5500mAh** | K50更大 |
| 快充功率 | 33W/67W | **67W** | K50统一 |
| CPU | 骁龙870 | 天玑8100 | 不同平台 |
| 机身重量 | 196g | 201g | 略有增加 |
| 出厂系统 | 澎湃OS 1.10 | MIUI 13/澎湃OS | 版本差异 |

## 适配修改清单

### 1. IslandStateManager.kt ✅

**修改内容**：
- 添加 `TURBO_CHARGING` 充电状态枚举（新增）
- 添加 `K50DeviceConfig` 设备配置对象
- 添加 `ChargingConfig` 充电配置对象
- 更新 `showCharging()` 方法支持涡轮快充参数
- 更新 `showFullBattery()` 显示5500mAh标识

**代码变更**：
```kotlin
// 新增状态
enum class IslandState {
    // ... 其他状态
    TURBO_CHARGING,  // K50新增：涡轮快充
}

// K50设备配置
object K50DeviceConfig {
    const val DEVICE_MODEL = " Redmi K50"
    const val HOLE_POSITION = HolePosition.CENTER  // 居中挖孔
    const val BATTERY_CAPACITY = 5500  // mAh
    const val FAST_CHARGING_POWER = 67  // Watt
}
```


```
│  ●                    [通知栏]     │
```

## 兼容性说明

### 向后兼容

K50适配版本保持了与K40版本的API兼容性：
- 核心状态管理接口不变
- 服务启动/停止接口不变
- 广播Action定义不变

### 特性兼容

新增的K50特性不会影响K40设备运行：
- `TURBO_CHARGING` 状态在K40上不会触发
- `islandPositionMode` 设置不影响K40显示
- 新增配置项有默认值保障兼容性

## 开发者备注

### 代码分支建议

```
main (K40版本)
  └── k50-adaptation (K50适配版本)
```

### 后续适配建议

如需适配其他设备：
1. 根据挖孔位置调整 `islandPositionMode`
2. 根据电池容量调整 `BATTERY_CAPACITY`
3. 根据快充功率调整 `FAST_CHARGING_POWER`
4. 更新 `HOLE_POSITION` 枚举值

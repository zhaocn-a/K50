# DynamicOasis-K50 适配说明

## 针对澎湃OS / MIUI 14的特殊处理

### 一、权限系统适配

澎湃OS/MIUI 14基于Android 13/14深度定制，权限管理系统与原生Android有较大差异。

#### 1.1 悬浮窗权限 (SYSTEM_ALERT_WINDOW)

**问题描述**：
澎湃OS/MIUI对悬浮窗权限管控严格，部分用户可能无法找到正确的授权入口。

**适配方案**：
```kotlin
// 提供多种跳转方式
fun requestOverlayPermission(context: Context) {
    val intents = listOf(
        // 方式1: 标准Android设置
        Intent(Settings.ACTION_MANAGE_OVERLAY_PERMISSION,
            Uri.parse("package:${context.packageName}")),
        
        // 方式2: MIUI特殊设置页
        Intent().apply {
            component = ComponentName(
                "com.miui.securitycenter",
                "com.miui.permcenter.autostart.AutoStartManagementActivity"
            )
        },
        
        // 方式3: 应用详情页
        Intent(Settings.ACTION_APPLICATION_DETAILS_SETTINGS,
            Uri.parse("package:${context.packageName}"))
    )
    
    // 尝试每个intent，找到可用的
    for (intent in intents) {
        if (intent.resolveActivity(context.packageManager) != null) {
            context.startActivity(intent)
            break
        }
    }
}
```

#### 1.2 通知访问权限 (NotificationListenerService)

**问题描述**：
澎湃OS的通知过滤机制可能导致部分通知无法被捕获。

**适配方案**：
```kotlin
class NotificationInterceptorService : NotificationListenerService() {
    
    // 澎湃OS/MIUI特定的包名过滤
    private val hyperOSNotificationFilters = listOf(
        "com.android.systemui",
        "com.miui.home",
        "com.miui.securitycenter",
        "com.miui.powerkeeper",
        "com.xiaomi.miservices"
    )
    
    override fun onNotificationPosted(sbn: StatusBarNotification?) {
        // 检查是否是系统通知
        val packageName = sbn?.packageName ?: return
        
        // 澎湃OS/MIUI特殊处理：部分系统通知需要特殊处理
        if (isHyperOSSystemNotification(packageName)) {
            handleHyperOSNotification(sbn)
            return
        }
        
        super.onNotificationPosted(sbn)
    }
    
    private fun isHyperOSSystemNotification(packageName: String): Boolean {
        return hyperOSNotificationFilters.any { 
            packageName.startsWith(it) 
        }
    }
}
```

#### 2.2 电量优化适配

澎湃OS/MIUI的省电策略会自动限制后台应用，需要引导用户添加白名单：

```kotlin
fun requestBatteryOptimizationExemption(context: Context) {
    if (!isIgnoringBatteryOptimizations(context)) {
        AlertDialog.Builder(context)
            .setTitle("省电设置")
            .setMessage("为了确保灵动岛正常工作，请将应用设置为\"无限制\"或\"不优化\"。")
            .setPositiveButton("去设置") { _, _ ->
                try {
                    val intent = Intent().apply {
                        component = ComponentName(
                            "com.miui.powerkeeper",
                            "com.miui.powerkeeper.ui.HiddenAppsConfigActivity"
                        )
                        putExtra("package_name", context.packageName)
                        putExtra("package_label", getAppName(context))
                    }
                    context.startActivity(intent)
                } catch (e: Exception) {
                    // 回退到标准设置
                    context.startActivity(
                        Intent(Settings.ACTION_IGNORE_BATTERY_OPTIMIZATION_SETTINGS)
                    )
                }
            }
            .show()
    }
}
```

### 三、开机自启适配

澎湃OS/MIUI的开机自启需要额外的权限：

#### 3.1 检测自启权限状态

```kotlin
fun isAutoStartEnabled(context: Context): Boolean {
    return try {
        val autoStartIntent = Intent().apply {
            component = ComponentName(
                "com.miui.securitycenter",
                "com.miui.permcenter.autostart.AutoStartManagementActivity"
            )
        }
        autoStartIntent.resolveActivity(context.packageManager) != null
    } catch (e: Exception) {
        false
    }

```

```kotlin
fun requestAutoStartPermission(context: Context) {
```

```kotlin
    // 使用节流阀避免频繁更新
    private var lastUpdateTime = 0L

    override fun onReceive(context: Context, intent: Intent) {
        val currentTime = System.currentTimeMillis()
        if (currentTime - lastUpdateTime < updateInterval) {
            return

        // 处理电池状态...
        handleBatteryIntent(intent, context)
    }

        val status = intent.getIntExtra("status", 0)

                            plugged == BatteryManager.BATTERY_PLUGGED_AC
        
        // 更新状态...
    }
}
```

### 五、红米K50居中挖孔屏适配

红米K50采用居中挖孔屏设计，前置摄像头位于屏幕顶部中央。

#### 5.1 灵动岛居中定位

```kotlin
private fun createLayoutParams(): WindowManager.LayoutParams {
    val density = resources.displayMetrics.density
    val width = (280 * density).toInt()
    val height = (50 * density).toInt()
    
    return WindowManager.LayoutParams(
        width,
        height,
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O)
            WindowManager.LayoutParams.TYPE_APPLICATION_OVERLAY
        else
            WindowManager.LayoutParams.TYPE_PHONE,
        WindowManager.LayoutParams.FLAG_NOT_FOCUSABLE or
                WindowManager.LayoutParams.FLAG_LAYOUT_NO_LIMITS,
        PixelFormat.TRANSLUCENT
    ).apply {
        gravity = Gravity.TOP or Gravity.CENTER_HORIZONTAL  // K50居中定位
        x = 0  // 无需X轴偏移
        y = (prefs.positionY * density).toInt().coerceAtLeast((30 * density).toInt())
    }
}
```

### 六、充电功率适配

红米K50支持67W快充，需要相应调整充电状态检测逻辑。

#### 6.1 67W快充检测

```kotlin
/**
 * 红米K50充电配置
 * 电池容量：5500mAh
 * 快充功率：67W
 * 充电时间：约48分钟充满
 */
object K50ChargingConfig {
    const val NORMAL_CHARGING = 0      // 普通充电 (<15W)
    const val FAST_CHARGING = 1        // 快速充电 (15W-45W)
    const val TURBO_CHARGING = 2       // 涡轮快充 (45W-67W)
    
    // K50支持67W快充，使用电荷泵技术
    const val EXPECTED_MAX_POWER = 67  // Watt
    const val BATTERY_CAPACITY = 5500  // mAh
    
    // 充电速度等级
    fun getChargingLevel(plugged: Int, voltage: Int, current: Int): Int {
        val power = (voltage * current) / 1000 // 简化的功率计算
        
        return when {
            power >= 45 -> TURBO_CHARGING
            power >= 15 -> FAST_CHARGING
            else -> NORMAL_CHARGING
        }
    }
}
```

### 七、K50与K40适配差异对比

| 适配项 | K40 (澎湃OS 1.10) | K50 (MIUI 13/澎湃OS) | 差异说明 |
|--------|-------------------|---------------------|----------|
| 挖孔位置 | 左侧 | **居中** | K50灵动岛居中 |
| 屏幕分辨率 | 2400×1080 | **3200×1440** | K50为2K屏 |
| 电池容量 | 4520mAh | **5500mAh** | K50更大电池 |
| 快充功率 | 33W/67W | **67W** | K50统一67W |
| 灵动岛X偏移 | 需要偏左 | **无需偏移** | K50居中挖孔 |
| 系统版本 | 澎湃OS 1.10 | MIUI 13/澎湃OS | 适配逻辑一致 |
| 摄像头 | 左侧2000万 | **居中2000万** | K50居中设计 |
| 前置传感器 | IMX596 | IMX596 | 相同传感器 |

### 八、注意事项

1. **居中挖孔设计**：K50前置摄像头居中，灵动岛默认居中显示，无需横向偏移
2. **更大电池**：5500mAh电池意味着充电时间更长，67W快充约48分钟充满
3. **2K屏幕**：K50的2K屏幕显示效果更细腻，像素密度526ppi
4. **MIUI 13**：K50出厂系统为MIUI 13，部分功能可能随系统更新变化
5. **澎湃OS升级**：部分K50已推送澎湃OS升级，适配逻辑基本一致

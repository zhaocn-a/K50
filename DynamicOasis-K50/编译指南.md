# 编译指南

## 环境要求

- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Gradle 8.0 (项目已包含 wrapper)

## 快速开始

### 方式一：使用 Android Studio

1. 打开 Android Studio
2. 选择 "Open an existing project"
3. 选择 `开发项目/DynamicOasis-K40` 文件夹
4. 等待 Gradle 同步完成（约5-10分钟）
5. 点击 Run > Run 'app' 或使用快捷键 Shift + F10

### 方式二：使用命令行

```bash
cd 开发项目/DynamicOasis-K40

# 赋予执行权限
chmod +x gradlew

# 同步依赖
./gradlew dependencies --refresh-dependencies

# 构建 Debug 版本
./gradlew assembleDebug

# 构建 Release 版本
./gradlew assembleRelease
```

## 输出位置

### Debug APK

app/build/outputs/apk/debug/dynamicoasis-k40-hyperos-debug.apk
```

### Release APK
```
app/build/outputs/apk/release/dynamicoasis-k40-hyperos-release.apk

## 签名配置

Release 版本需要签名配置。在 `app/build.gradle` 中添加：

```groovy
android {
    signingConfigs {
        release {
            storeFile file("your-keystore.jks")
            storePassword "your-password"
            keyAlias "your-alias"
            keyPassword "your-password"
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
        }
    }
}
```

## 常见问题

### 1. Gradle 同步失败

**问题**: 依赖下载失败或超时

**解决方案**:
```bash
./gradlew --stop
rm -rf ~/.gradle/caches
./gradlew assembleDebug --refresh-dependencies
```

### 2. SDK 找不到

**问题**: `ANDROID_HOME` 未设置

**解决方案**:
在 `local.properties` 中设置 SDK 路径：
```properties
sdk.dir=/path/to/your/android-sdk
```

或在环境变量中设置：
```bash
export ANDROID_HOME=/path/to/your/android-sdk
```

### 3. Java 版本不兼容

**问题**: `Unsupported class file major version`

**解决方案**:
确保使用 JDK 17：
```bash
export JAVA_HOME=/path/to/jdk17
java -version
```

### 4. 编译内存不足

**解决方案**:
在 `gradle.properties` 中增加内存：
```properties
org.gradle.jvmargs=-Xmx4096m -Dfile.encoding=UTF-8
```
